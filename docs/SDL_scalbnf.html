SDL_scalbnf

Scale x by an integer power of two.

Header File

Defined in <SDL3/SDL_stdinc.h>

Syntax

float SDL_scalbnf(float x, int n);

Function Parameters
float x floating point value to be scaled.
int n integer exponent.

Return Value

(float) Returns x * 2^n.

Remarks

Multiplies x by the nth power of the floating point radix (always 2).

Domain: -INF <= x <= INF, n integer

Range: -INF <= y <= INF

This function operates on single-precision floating point values, use SDL_scalbn for double-precision floats.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
