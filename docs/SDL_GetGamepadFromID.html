SDL_GetGamepadFromID

Get the SDL_Gamepad associated with a joystick instance ID, if it has been opened.

Header File

Defined in <SDL3/SDL_gamepad.h>

Syntax

SDL_Gamepad * SDL_GetGamepadFromID(SDL_JoystickID instance_id);

Function Parameters
SDL_JoystickID instance_id the joystick instance ID of the gamepad.

Return Value

(SDL_Gamepad *) Returns an SDL_Gamepad on success or NULL on failure or if it hasn't been opened yet; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
