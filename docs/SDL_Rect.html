SDL_Rect

A rectangle, with the origin at the upper left (using integers).

Header File

Defined in <SDL3/SDL_rect.h>

Syntax

typedef struct SDL_Rect
{
    int x, y;
    int w, h;
} SDL_Rect;

Version

This struct is available since SDL 3.2.0.
