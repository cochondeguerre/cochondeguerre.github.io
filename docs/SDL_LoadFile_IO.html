SDL_LoadFile_IO

Load all the data from an SDL data stream.

Header File

Defined in <SDL3/SDL_iostream.h>

Syntax

void * SDL_LoadFile_IO(SDL_IOStream *src, size_t *datasize, bool closeio);

Function Parameters
SDL_IOStream * src the SDL_IOStream to read all available data from.
size_t * datasize a pointer filled in with the number of bytes read, may be NULL.
bool closeio if true, calls SDL_CloseIO() on src before returning, even in the case of an error.

Return Value

(void *) Returns the data or NULL on failure; call SDL_GetError() for more information.

Remarks

The data is allocated with a zero byte at the end (null terminated) for convenience. This extra byte is not included in the value reported via datasize.

The data should be freed with SDL_free().

Thread Safety

Do not use the same SDL_IOStream from two threads at once.

Version

This function is available since SDL 3.2.0.
