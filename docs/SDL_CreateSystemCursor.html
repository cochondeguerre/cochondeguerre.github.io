SDL_CreateSystemCursor

Create a system cursor.

Header File

Defined in <SDL3/SDL_mouse.h>

Syntax

SDL_Cursor * SDL_CreateSystemCursor(SDL_SystemCursor id);

Function Parameters
SDL_SystemCursor id an SDL_SystemCursor enum value.

Return Value

(SDL_Cursor *) Returns a cursor on success or NULL on failure; call SDL_GetError() for more information.

Thread Safety

This function should only be called on the main thread.

Version

This function is available since SDL 3.2.0.

Code Examples

SDL_Cursor* cursor;
cursor = SDL_CreateSystemCursor(SDL_SYSTEM_CURSOR_POINTER);
SDL_SetCursor(cursor);
