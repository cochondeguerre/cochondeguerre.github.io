SDL_WindowID

This is a unique ID for a window.

Header File

Defined in <SDL3/SDL_video.h>

Syntax

typedef Uint32 SDL_WindowID;

Remarks

The value 0 is an invalid ID.

Version

This datatype is available since SDL 3.2.0.
