SDL_FPoint

The structure that defines a point (using floating point values).

Header File

Defined in <SDL3/SDL_rect.h>

Syntax

typedef struct SDL_FPoint
{
    float x;
    float y;
} SDL_FPoint;

Version

This struct is available since SDL 3.2.0.
