SDL_NS_PER_US

Number of nanoseconds in a microsecond.

Header File

Defined in <SDL3/SDL_timer.h>

Syntax

#define SDL_NS_PER_US       1000

Remarks

This is always 1000.

Version

This macro is available since SDL 3.2.0.
