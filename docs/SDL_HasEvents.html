SDL_HasEvents

Check for the existence of certain event types in the event queue.

Header File

Defined in <SDL3/SDL_events.h>

Syntax

bool SDL_HasEvents(Uint32 minType, Uint32 maxType);

Function Parameters
Uint32 minType the low end of event type to be queried, inclusive; see SDL_EventType for details.
Uint32 maxType the high end of event type to be queried, inclusive; see SDL_EventType for details.

Return Value

(bool) Returns true if events with type >= minType and <= maxType are present, or false if not.

Remarks

If you need to check for a single event type, use SDL_HasEvent() instead.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
