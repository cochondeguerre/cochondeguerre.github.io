Uint8

An unsigned 8-bit integer type.

Header File

Defined in <SDL3/SDL_stdinc.h>

Syntax

#define SDL_MAX_UINT8   ((Uint8)0xFF)           /* 255 */
#define SDL_MIN_UINT8   ((Uint8)0x00)           /* 0 */

Version
