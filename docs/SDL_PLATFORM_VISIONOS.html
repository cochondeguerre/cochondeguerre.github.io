SDL_PLATFORM_VISIONOS

A preprocessor macro that is only defined if compiling for visionOS.

Header File

Defined in <SDL3/SDL_platform_defines.h>

Syntax

#define SDL_PLATFORM_VISIONOS 1

Version

This macro is available since SDL 3.2.0.
