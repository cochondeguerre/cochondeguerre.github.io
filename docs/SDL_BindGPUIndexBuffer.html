SDL_BindGPUIndexBuffer

Binds an index buffer on a command buffer for use with subsequent draw calls.

Header File

Defined in <SDL3/SDL_gpu.h>

Syntax

void SDL_BindGPUIndexBuffer(
    SDL_GPURenderPass *render_pass,
    const SDL_GPUBufferBinding *binding,
    SDL_GPUIndexElementSize index_element_size);

Function Parameters
SDL_GPURenderPass * render_pass a render pass handle.
const SDL_GPUBufferBinding * binding a pointer to a struct containing an index buffer and offset.
SDL_GPUIndexElementSize index_element_size whether the index values in the buffer are 16- or 32-bit.

Version

This function is available since SDL 3.2.0.
