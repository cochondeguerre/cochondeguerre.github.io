SDL_PLATFORM_BSDI

A preprocessor macro that is only defined if compiling for BSDi

Header File

Defined in <SDL3/SDL_platform_defines.h>

Syntax

#define SDL_PLATFORM_BSDI 1

Version

This macro is available since SDL 3.2.0.
