SDL_GetRenderOutputSize

Get the output size in pixels of a rendering context.

Header File

Defined in <SDL3/SDL_render.h>

Syntax

bool SDL_GetRenderOutputSize(SDL_Renderer *renderer, int *w, int *h);

Function Parameters
SDL_Renderer * renderer the rendering context.
int * w a pointer filled in with the width in pixels.
int * h a pointer filled in with the height in pixels.

Return Value

(bool) Returns true on success or false on failure; call SDL_GetError() for more information.

Remarks

This returns the true output size in pixels, ignoring any render targets or logical size and presentation.

For the output size of the current rendering target, with logical size adjustments, use SDL_GetCurrentRenderOutputSize() instead.

Thread Safety

This function should only be called on the main thread.

Version

This function is available since SDL 3.2.0.
