SDL_GetGPURendererDevice

Return the GPU device used by a renderer.

Header File

Defined in <SDL3/SDL_render.h>

Syntax

SDL_GPUDevice * SDL_GetGPURendererDevice(SDL_Renderer *renderer);

Function Parameters
SDL_Renderer * renderer the rendering context.

Return Value

(SDL_GPUDevice *) Returns the GPU device used by the renderer, or NULL if the renderer is not a GPU renderer; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.4.0.
