Sint16

A signed 16-bit integer type.

Header File

Defined in <SDL3/SDL_stdinc.h>

Syntax

typedef int16_t Sint16;
#define SDL_MAX_SINT16  ((Sint16)0x7FFF)        /* 32767 */
#define SDL_MIN_SINT16  ((Sint16)(~0x7FFF))     /* -32768 */

Version

This macro is available since SDL 3.2.0.
