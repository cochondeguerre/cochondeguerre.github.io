SDL_OpenFileStorage

Opens up a container for local filesystem storage.

Header File

Defined in <SDL3/SDL_storage.h>

Syntax

SDL_Storage * SDL_OpenFileStorage(const char *path);

Function Parameters
const char * path the base path prepended to all storage paths, or NULL for no base path.

Return Value

(SDL_Storage *) Returns a filesystem storage container on success or NULL on failure; call SDL_GetError() for more information.

Remarks

This is provided for development and tools. Portable applications should use SDL_OpenTitleStorage() for access to game data and SDL_OpenUserStorage() for access to user data.

Version

This function is available since SDL 3.2.0.
