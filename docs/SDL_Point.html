SDL_Point

The structure that defines a point (using integers).

Header File

Defined in <SDL3/SDL_rect.h>

Syntax

typedef struct SDL_Point
{
    int x;
    int y;
} SDL_Point;

Version

This struct is available since SDL 3.2.0.
