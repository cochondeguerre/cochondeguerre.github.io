SDL_HAPTIC_SINE

Sine wave effect supported.

Header File

Defined in <SDL3/SDL_haptic.h>

Syntax

#define SDL_HAPTIC_SINE         (1u<<1)

Remarks

Periodic haptic effect that simulates sine waves.

Version

This macro is available since SDL 3.2.0.
