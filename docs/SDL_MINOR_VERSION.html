SDL_MINOR_VERSION

The current minor version of the SDL headers.

Header File

Defined in <SDL3/SDL_version.h>

Syntax

#define SDL_MINOR_VERSION   5

Remarks

If this were SDL version 3.2.1, this value would be 2.

Version

This macro is available since SDL 3.2.0.
