SDL_GetJoystickFromID

Get the SDL_Joystick associated with an instance ID, if it has been opened.

Header File

Defined in <SDL3/SDL_joystick.h>

Syntax

SDL_Joystick * SDL_GetJoystickFromID(SDL_JoystickID instance_id);

Function Parameters
SDL_JoystickID instance_id the instance ID to get the SDL_Joystick for.

Return Value

(SDL_Joystick *) Returns an SDL_Joystick on success or NULL on failure or if it hasn't been opened yet; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
