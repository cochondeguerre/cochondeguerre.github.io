SDL_GetCameraID

Get the instance ID of an opened camera.

Header File

Defined in <SDL3/SDL_camera.h>

Syntax

SDL_CameraID SDL_GetCameraID(SDL_Camera *camera);

Function Parameters
SDL_Camera * camera an SDL_Camera to query.

Return Value

(SDL_CameraID) Returns the instance ID of the specified camera on success or 0 on failure; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
