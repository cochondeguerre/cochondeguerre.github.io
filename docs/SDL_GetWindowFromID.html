SDL_GetWindowFromID

Get a window from a stored ID.

Header File

Defined in <SDL3/SDL_video.h>

Syntax

SDL_Window * SDL_GetWindowFromID(SDL_WindowID id);

Function Parameters
SDL_WindowID id the ID of the window.

Return Value

(SDL_Window *) Returns the window associated with id or NULL if it doesn't exist; call SDL_GetError() for more information.

Remarks

The numeric ID is what SDL_WindowEvent references, and is necessary to map these events to specific SDL_Window objects.

Thread Safety

This function should only be called on the main thread.

Version

This function is available since SDL 3.2.0.
