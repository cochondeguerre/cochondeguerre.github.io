SDL_GPUTextureSupportsSampleCount

Determines if a sample count for a texture format is supported.

Header File

Defined in <SDL3/SDL_gpu.h>

Syntax

bool SDL_GPUTextureSupportsSampleCount(
    SDL_GPUDevice *device,
    SDL_GPUTextureFormat format,
    SDL_GPUSampleCount sample_count);

Function Parameters
SDL_GPUDevice * device a GPU context.
SDL_GPUTextureFormat format the texture format to check.
SDL_GPUSampleCount sample_count the sample count to check.

Return Value

(bool) Returns whether the sample count is supported for this texture format.

Version

This function is available since SDL 3.2.0.
