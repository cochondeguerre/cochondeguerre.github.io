SDL_GetJoystickFromPlayerIndex

Get the SDL_Joystick associated with a player index.

Header File

Defined in <SDL3/SDL_joystick.h>

Syntax

SDL_Joystick * SDL_GetJoystickFromPlayerIndex(int player_index);

Function Parameters
int player_index the player index to get the SDL_Joystick for.

Return Value

(SDL_Joystick *) Returns an SDL_Joystick on success or NULL on failure; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
