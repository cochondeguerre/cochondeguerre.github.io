SDL_StopHapticEffect

Stop the haptic effect on its associated haptic device.

Header File

Defined in <SDL3/SDL_haptic.h>

Syntax

bool SDL_StopHapticEffect(SDL_Haptic *haptic, SDL_HapticEffectID effect);

Function Parameters
SDL_Haptic * haptic the SDL_Haptic device to stop the effect on.
SDL_HapticEffectID effect the ID of the haptic effect to stop.

Return Value

(bool) Returns true on success or false on failure; call SDL_GetError() for more information.

Version

This function is available since SDL 3.2.0.
