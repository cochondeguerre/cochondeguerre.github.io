SDL_GetVideoDriver

Get the name of a built in video driver.

Header File

Defined in <SDL3/SDL_video.h>

Syntax

const char * SDL_GetVideoDriver(int index);

Function Parameters
int index the index of a video driver.

Return Value

(const char *) Returns the name of the video driver with the given index, or NULL if index is out of bounds.

Remarks

The video drivers are presented in the order in which they are normally checked during initialization.

The names of drivers are all simple, low-ASCII identifiers, like "cocoa", "x11" or "windows". These never have Unicode characters, and are not meant to be proper names.

Thread Safety

This function should only be called on the main thread.

Version

This function is available since SDL 3.2.0.
