SDL_GetCameraProperties

Get the properties associated with an opened camera.

Header File

Defined in <SDL3/SDL_camera.h>

Syntax

SDL_PropertiesID SDL_GetCameraProperties(SDL_Camera *camera);

Function Parameters
SDL_Camera * camera the SDL_Camera obtained from SDL_OpenCamera().

Return Value

(SDL_PropertiesID) Returns a valid property ID on success or 0 on failure; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
