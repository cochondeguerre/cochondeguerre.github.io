SDL_AudioDeviceID

SDL Audio Device instance IDs.

Header File

Defined in <SDL3/SDL_audio.h>

Syntax

typedef Uint32 SDL_AudioDeviceID;

Remarks

Zero is used to signify an invalid/null device.

Version

This datatype is available since SDL 3.2.0.
