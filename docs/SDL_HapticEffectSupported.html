SDL_HapticEffectSupported

Check to see if an effect is supported by a haptic device.

Header File

Defined in <SDL3/SDL_haptic.h>

Syntax

bool SDL_HapticEffectSupported(SDL_Haptic *haptic, const SDL_HapticEffect *effect);

Function Parameters
SDL_Haptic * haptic the SDL_Haptic device to query.
const SDL_HapticEffect * effect the desired effect to query.

Return Value

(bool) Returns true if the effect is supported or false if it isn't.

Version

This function is available since SDL 3.2.0.
