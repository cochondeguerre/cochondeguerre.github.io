SDL_WPRINTF_VARARG_FUNCV

Macro that annotates a va_list function that operates like wprintf.

Header File

Defined in <SDL3/SDL_stdinc.h>

Syntax

#define SDL_WPRINTF_VARARG_FUNCV( fmtargnumber ) /* __attribute__ (( format( __wprintf__, fmtargnumber, 0 ))) */

Remarks

If we were to annotate vfwprintf:

int vfwprintf(FILE *f, const wchar_t *fmt, va_list ap) SDL_WPRINTF_VARARG_FUNC(2);

This notes that the second parameter should be a wprintf-style format wide string, followed by a va_list. The compiler or other analysis tools can warn when this doesn't appear to be the case.

On compilers without this annotation mechanism, this is defined to nothing.

This can (and should) be used with SDL_PRINTF_FORMAT_STRING as well, which between them will cover at least Visual Studio, GCC, and Clang.

Version

This macro is available since SDL 3.2.0.
