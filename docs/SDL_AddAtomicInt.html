SDL_AddAtomicInt

Add to an atomic variable.

Header File

Defined in <SDL3/SDL_atomic.h>

Syntax

int SDL_AddAtomicInt(SDL_AtomicInt *a, int v);

Function Parameters
SDL_AtomicInt * a a pointer to an SDL_AtomicInt variable to be modified.
int v the desired value to add.

Return Value

(int) Returns the previous value of the atomic variable.

Remarks

This function also acts as a full memory barrier.

Note: If you don't know what this function is for, you shouldn't use it!

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
