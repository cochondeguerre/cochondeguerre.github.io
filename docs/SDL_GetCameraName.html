SDL_GetCameraName

Get the human-readable device name for a camera.

Header File

Defined in <SDL3/SDL_camera.h>

Syntax

const char * SDL_GetCameraName(SDL_CameraID instance_id);

Function Parameters
SDL_CameraID instance_id the camera device instance ID.

Return Value

(const char *) Returns a human-readable device name or NULL on failure; call SDL_GetError() for more information.

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
