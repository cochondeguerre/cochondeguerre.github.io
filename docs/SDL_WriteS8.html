SDL_WriteS8

Use this function to write a signed byte to an SDL_IOStream.

Header File

Defined in <SDL3/SDL_iostream.h>

Syntax

bool SDL_WriteS8(SDL_IOStream *dst, Sint8 value);

Function Parameters
SDL_IOStream * dst the SDL_IOStream to write to.
Sint8 value the byte value to write.

Return Value

(bool) Returns true on successful write or false on failure; call SDL_GetError() for more information.

Thread Safety

Do not use the same SDL_IOStream from two threads at once.

Version

This function is available since SDL 3.2.0.
