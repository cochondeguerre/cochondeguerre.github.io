SDL_UpdateJoysticks

Update the current state of the open joysticks.

Header File

Defined in <SDL3/SDL_joystick.h>

Syntax

void SDL_UpdateJoysticks(void);

Remarks

This is called automatically by the event loop if any joystick events are enabled and SDL_HINT_AUTO_UPDATE_JOYSTICKS hasn't been set to "0".

Thread Safety

It is safe to call this function from any thread.

Version

This function is available since SDL 3.2.0.
